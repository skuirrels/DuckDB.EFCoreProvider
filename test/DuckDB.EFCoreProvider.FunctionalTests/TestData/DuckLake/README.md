# DuckLake v0.3 migration fixture

`v03.db.gz.b64` is a Base64 representation of the compressed `v03.db.gz` migration fixture from the
[DuckDB DuckLake repository](https://github.com/duckdb/ducklake/blob/3108822ddfb98288360cb71d173d62fadab26f38/data/old_ducklake/v03.db.gz).
It is decoded and decompressed only into a per-test temporary directory. The compressed fixture SHA-256 is
`e51404e176a06ab0209e0c4697df9b9f170e81f02c5dcf624c192a3f34ed2309`.
